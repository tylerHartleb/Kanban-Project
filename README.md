## Usage

To run the app in your local environment run the following commands:

```bash
  # Run the app
  $ npm run start
```

To build the app run the following commands:

```bash
  # Build the app
  $ npm run build
```

To run the unit tests run the following commands:

```bash
  # Test command
  $ npm run test
```
